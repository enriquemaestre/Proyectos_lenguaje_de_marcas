Hipervínculos o hipertextos en HTML5

Los hipertextos son elementos caracteristicos de HTML5 que nos permite que las páginas web puedan
relacionarse entre sí.
Los hipervínculos están formados por dos partes:
  • El enlace: es el elemento sobre el que se puede hacer click con el ratón.
  • El destino: es el elemento que se muestra al hacer click en el enlace.

Los hipervínculos tienen la siguiente estructura:
  • La etiqueta de apertura y cierre a(anchor)
  • tiene un atributo href, es donde vamos a poner el link del elemento que queremos como destino
  • Entre la etiqueta de apertura y la de cierre de a, el texto del hipervinculo o hiperenlace
